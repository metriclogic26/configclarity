ConfigClarity is a free network of browser-based sysadmin audit tools. Tools include: Cron Job Overlap Checker and Visualiser, SSL Certificate Expiry Checker, Docker Compose Auditor, UFW Firewall Auditor, Reverse Proxy Flow Mapper, and robots.txt Validator. No signup required. No backend. All processing runs client-side in the browser. Nothing is stored or transmitted.
